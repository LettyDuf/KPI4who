Synthèse de l'expertise UX/UI consultée

Principe directeur retenu : bordure gauche 4px pleine de la teinte dominante (via ::before absolu pour garder les coins arrondis) + fond de bandeau très léger à l'état fermé. À l'ouverture, le fond passe au blanc neutre (l'ombre + la bordure bleu-moyen du design system prennent le relais), mais le filet gauche coloré reste pour ancrer la lecture verticale même pendant le scroll.

Raisonnements sémantiques

Violet = réflexion / boucle itérative ; éloigné des bleus hiérarchiques. Brun taupe = matière, atelier, héritage industriel Lean, chaleur tellurique. Cyan-teal = tech / instrumentation, sans empiéter sur les bleus de niveau. Bleu-gris ardoise = direction sobre, cockpit stratégique. Magenta-carmin = incidents / exploitation, signal sans taper dans l'orange accent. Olive-ocre = cartographie, sensemaking, terrain mouvant (Cynefin). Gris bleuté = neutralité assumée pour le générique (sans fond coloré, il se fond dans le fond général).

Chevron renforcé

SVG inline (plus net qu'unicode), disque de 1.75rem, trait épais 3px, couleur --niv-1 au lieu du gris précédent. Ratio de contraste ≈ 12:1 sur fond #eceff1. Transition cubic-bezier avec léger overshoot pour un feedback plus expressif. À l'ouverture, le disque bascule sur --niv-1-clair avec bordure --niv-1-moyen.

Signaux d'alerte à surveiller au rendu

1. Effet « bonbonnière » : sept bandeaux fermés empilés peuvent virer envahissant même avec des fonds pâles. Si ça agresse à l'œil, basculer 4 familles sur 7 vers « bordure gauche seule » (sans fond). Ancres chromatiques à garder : Empirisme agile, Excellence opérationnelle, Universel.

2. Collision avec la porte « Par mon niveau » : la bordure ardoise de Pilotage par les objectifs pourrait se confondre avec une teinte de niveau si on ouvre les deux portes côte à côte. À tester en usage réel.

3. Daltonisme deutéran / protan : magenta (ITSM) et brun taupe (Lean) se rapprochent pour certains types de vision. La bordure gauche + le libellé texte restent les canaux porteurs, la couleur n'est qu'une aide mnémonique.
